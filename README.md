# Pello

Mobile Project Management System


## Run Locally

Clone the project

```bash
  git clone https://github.com/reygranda/PelloMobileApp.git
```

Go to the project directory

```bash
  cd PelloMobileApp
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  expo start
```
